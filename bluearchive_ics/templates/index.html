📅 BlueArchive.ics

蔚蓝档案活动日历订阅服务

将《蔚蓝档案》的卡池、活动和总力战信息转换为标准 iCalendar (.ics) 日历文件，可导入任何日历应用。

🚀 使用方法

订阅日历链接：

完整地址：加载中... 
GET /ba.ics

📋 参数说明

参数 类型 必填 说明
server string ❌ 区服：cn 国服（默认）/ in 国际服 / jp 日服
filter string ❌ 事件类型筛选，可重复。可选值：assault 总力战 / event 活动 / card 卡池
