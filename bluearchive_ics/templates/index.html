蔚蓝档案活动日历订阅服务

将《蔚蓝档案》的卡池、活动和总力战信息转换为标准 iCalendar (.ics) 日历文件，可导入任何日历应用。

🚀 使用方法

订阅日历链接：

完整地址：加载中...

📋 路由格式

  • /{server}/{type}.ics — 指定区服和类型
  • /{type}.ics — 默认国服
  • /ba.ics — 兼容旧链接

📋 参数说明

参数 可选值 说明
server cn / in / jp 区服：国服（默认）/ 国际服 / 日服
type ba / all / card / assault / event 事件类型：全部 / 全部 / 卡池 / 总力战 / 活动

🔀 卡池合并

相同开始和结束时间的卡池会自动合并为一条显示。如需查看完整卡池信息，请使用 /full-card-info.ics 终结点。
